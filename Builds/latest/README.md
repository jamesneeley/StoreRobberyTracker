Store Robbery Enhanced — Latest Build
Version: v2.6.7.9 (Latest release)

Thank you for downloading the latest build of Store Robbery Enhanced.
This mod brings the GTA Online store robbery achievement experience into
GTA V Single Player with expanded mechanics, smarter AI, and a complete
robbery ecosystem.

FEATURES
--------
- 20 fully supported stores (19 Online stores + Ace Liquor)
- Clerk AI: fear, surrender, flee, escalation
- Register payouts and alarm escalation
- SafeCrack minigame with UI and difficulty scaling
- Cooldown system to prevent farming
- Subtitle prompts inspired by GTA Online
- Minimap-only blip system with dynamic state colors
- Full debug suite (overlay, camera debug, logging)

INSTALLATION
------------
1. Install ScriptHookV.
2. Install ScriptHookVDotNet 3.9.0 Enhanced.
3. Place StoreRobberTracker.dll into:
   Grand Theft Auto V/scripts/

NOTES
-----
- Store 8 is the official baseline reference store.
- All store IDs are finalized and stable.
- This build completes Phase 3 and prepares the project for Phase 4.
