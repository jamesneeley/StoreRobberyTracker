Store Robbery Enhanced — v2.6.7.9
Initial Release

This is the first official public release of Store Robbery Enhanced, a complete
Single Player recreation and expansion of the GTA Online store robbery
achievement experience.

FEATURES INCLUDED IN THIS RELEASE
---------------------------------
- Full 21‑store dataset (20 Online stores + Ace Liquor)
- Store ID remapping finalized (Old Store 14 → New Store 8)
- Complete robbery system:
    - Register payouts
    - Clerk reactions
    - Alarm escalation
    - Cooldowns
    - Subtitles
- Clerk AI overhaul:
    - Fear, surrender, flee, escalation
- SafeCrack minigame with UI and difficulty scaling
- Minimap‑only blip system with dynamic state colors
- Full debug suite:
    - Debug overlay
    - Camera debug
    - Logging system
- Modular architecture:
    - Config
    - Data
    - Systems
    - UI
    - Minigame
    - Initialization
    - Debug

INSTALLATION
------------
1. Install ScriptHookV.
2. Install ScriptHookVDotNet 3.9.0 Enhanced.
3. Place StoreRobberyTracker.dll into:
   Grand Theft Auto V/scripts/

NOTES
-----
- Store 8 is the official baseline reference store.
- All store IDs are finalized and stable.
- This build completes Phase 3 and prepares the project for Phase 4.
